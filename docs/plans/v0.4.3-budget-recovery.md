## 2. 数据库 Schema

### 2.1 新表 `budgets`

```sql
CREATE TABLE IF NOT EXISTS budgets (
    id              INTEGER PRIMARY KEY AUTOINCREMENT,
    scope_type      TEXT    NOT NULL,             -- 'user' | 'agent_kind' | 'global'
    scope_value     TEXT    NOT NULL,             -- user_id (str) / agent_kind / 'all'
    budget_type     TEXT    NOT NULL,             -- 'monthly_cost_usd' | 'monthly_tokens' | 'per_call_cost_usd'
    threshold       REAL    NOT NULL,
    action          TEXT    NOT NULL DEFAULT 'warn',  -- 'warn' | 'block'
    enabled         INTEGER DEFAULT 1,
    created_at      TEXT    DEFAULT (datetime('now','localtime')),
    updated_at      TEXT    DEFAULT (datetime('now','localtime')),
    UNIQUE (scope_type, scope_value, budget_type)
);
```

**关键设计**：
- 三层 scope（global / user / agent_kind）同表 DRY
- `budget_type` Literal 锁（`models/budget.py`）
- `action='warn'` 默认；`'block'` 仅允许 agent_kind per_call_cost_usd 用（**R-21 + R-S?-block-restriction**）
- UNIQUE 防重复 — **R-18 幂等基座**

### 2.2 messages 表不动（R-S6 守护）

System_Recovery 趋势仅读 v0.4.2 已有的 `recovery_attempt` 列，**0 schema 改动**。
列数稳定 24（v0.4.5+ 仍可评估迁移工具，但 v0.4.3 不触发）。

**流式（query-stream）SSE final 事件**：

## 5. 前端

### 5.1 ResultBlock budget banner（R-20 降噪）

```jsx
// budget_status === 'warn' 且 sessionStorage 未标记 → 显示橙色 banner
const [budgetWarnDismissed, setBudgetWarnDismissed] = useState(
  () => sessionStorage.getItem(`budget_warn_${user.id}_${YYYYMM}`) === '1'
);
const showBanner = budget_status === 'warn' && !budgetWarnDismissed;

{showBanner && (
  <div style={{ ...warnBannerStyle }}>
    ⚠️ 本月已用 {budget_meta.percentage}% 配额（${budget_meta.current.toFixed(2)} / ${budget_meta.threshold.toFixed(2)}）
    <button onClick={() => {
      sessionStorage.setItem(`budget_warn_${user.id}_${YYYYMM}`, '1');
      setBudgetWarnDismissed(true);
    }}>本会话不再提醒</button>
  </div>
)}
```

**R-20 实现策略**：
- 每个 user × 月份独立 sessionStorage key（避免跨月泄露）
- 用户主动关闭 → 本会话内不再弹（翻历史会话也不弹）
- 跨标签页/重启浏览器 → 重新提醒（sessionStorage 重置）
