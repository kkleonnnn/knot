## §A 项目背景（30 秒）

**KNOT** — FastAPI + React 自然语言 BI Agent，prod v0.6.2.1。

**生产 bug（Task #17）**：用户问"今日合约交易量和充值情况"（复合 metric — 一句问 2+ 指标）→ 返回 0/NULL。

**根因（前端确定）**：`MetricCard.jsx` 只渲染 `numericCols[0]`（首个数值列）：
```js
const valueCol = numericCols[0] || cols[0];  // ← 复合 metric SELECT SUM(volume), SUM(deposit) 只取第一个
```
复合 metric → 2 列 1 行 → 只显第一个，第二个丢弃；若首列聚合 0/NULL → 显 0。
