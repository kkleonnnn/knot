**Demo 与现实业务模型不兼容** — 需 **D2 双兼模式延伸**（视觉借用 + 业务字段 byte-equal）：

| 维度 | Demo 单值配置 | 现实 multi-scope CRUD |
|---|---|---|
| 数据模型 | 单组织全局预算单值 | `user × agent_kind × global` 三层独立预算（R-16 优先级 + R-23 实时） |
| 字段 | 月度上限 / 单次上限 / 告警阈值 / 默认模型 / 限流 | scope_type / scope_value / budget_type / threshold / action / enabled |
| 操作 | save / reset | CRUD（list / create / update / delete / toggle） |

**裁定**：**保业务 CRUD 模型 + 借 Demo 视觉语言**。Hero usage card placeholder + 视觉风格全面对齐（thead brandSoft 8% + mono labels + CSS Grid + ActionChip pattern）。

```jsx
function BudgetActionChip({ T, action }) {
  const color = action === 'block' ? T.warn : T.accent;  // warn=blocking; warn alert; accent=normal
  return (
    <span style={{
      color,
      background: `color-mix(in oklch, ${color} 12%, transparent)`,
      padding: '2px 8px',
      borderRadius: 4,
      fontWeight: 500,
      fontSize: 11,
      fontFamily: T.mono,
    }}>{action}</span>
  );
}
```

```jsx
function EnabledChip({ T, enabled, onClick }) {

}
```
