```jsx

```

**严禁** Unicode ↵ 字符；用纯 ASCII。

### Stage 2 R-261 focus-within React state 实施

```jsx
const [isFocused, setIsFocused] = useState(false);
// ...
<div style={{
  border: `1px solid ${isFocused ? T.accentSoft : T.inputBorder}`,
  transition: 'border-color 200ms, box-shadow 200ms',
  boxShadow: isFocused
    ? `${baseShadow}, 0 0 0 3px color-mix(in oklch, ${T.accent} 15%, transparent)`
    : baseShadow,
}}>
  <textarea onFocus={() => setIsFocused(true)} onBlur={() => setIsFocused(false)} ... />
</div>
```

inline style **无法直接用 `:focus-within` 伪类**，必须用 React state。
