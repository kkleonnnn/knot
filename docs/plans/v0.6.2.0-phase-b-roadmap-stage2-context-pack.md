### B.3 18+ 条新立红线（重点）

**B1 TOTP（10 条）**：Fernet 加密 / 不锁死账号 / admin 启动期降级 / 公测启动闸门 / 重置 audit / rate limit 5/min / 二次确认 / Fernet 路径 / **R-46-Tx 事务模式**（secret + recovery codes 同事务防部分写入）/ **R-27 race 100× 守护**（并发场景 rate limit 误差 ≤ 1）

**C2 HTTP 路由双层守护**：
- 正向白名单（catalog source:http + entity 明示）= 主路由
- 黑名单 regex（"历史/已平仓/强平/爆仓/ADL/(N天/N月/昨天/上周)" 等）= diagnostic warn 层（log 不阻断）

**A1 OOS-1 sustained 4 条**：无 tenant_id / 无跨 catalog 隔离 / 数据库连接共享 / audit_log 透明

**F4 AdminScreenTemplate PII 封装 3 条**：5 admin 屏 PII 三层防御（R-48/59/62）强制走模板层 / 不允许 bypass / 第 7 次远古激活复核前置
