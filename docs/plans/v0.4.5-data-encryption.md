## 1. 背景

v0.4.4 收官后 KNOT 进入「准生产」阶段。**当前 `bi_agent/data/bi_agent.db` 以明文存储 6 类敏感落点**，拖库即可直接复用受害者的 LLM 配额、打入业务库。v0.4.5 目标是把这 6 类落点全部迁到应用层加密，且保证：

1. 4 层契约不破（加解密只在 repositories）
2. 调用方零改动（services / api 见到的全是明文）
3. 启动 fail-fast（master key 缺失不允许 silent fallback）
4. 迁移幂等可恢复

### 1.1 敏感字段清单（最终确认 — 6 列）

| 表 | 字段 | 当前 schema | 数据来源 |
|---|---|---|---|
| `users` | `api_key` | `TEXT DEFAULT ''` | 用户级 LLM provider key |
| `users` | `openrouter_api_key` | `TEXT DEFAULT ''` | 用户级 OpenRouter key |
| `users` | `embedding_api_key` | `TEXT DEFAULT ''` | 用户级 embedding key |
| `users` | `doris_password` | `TEXT` | 用户业务库密码 |
| `data_sources` | `db_password` | `TEXT NOT NULL` | admin 配置数据源密码 |
| `app_settings` | value WHERE key IN (`SENSITIVE_KEYS`) | `TEXT` | 全局 OpenRouter key 等 |

`users.password_hash` 已是 bcrypt 不可逆 → **不加密**。
`SENSITIVE_KEYS` 白名单：`{openrouter_api_key, embedding_api_key}`（其他 setting 名留作扩展，见 §10 兼容矩阵）。

## 3. 技术选型

### 3.1 加密：Fernet（cryptography 库）

| 候选 | 优点 | 缺点 | 取舍 |
|---|---|---|---|
| **Fernet** | 标准化 / 自带 IV+HMAC+timestamp / 使用面无 footgun | 算法定死 AES-128-CBC + HMAC-SHA256 | ✅ 选 |
| AES-GCM raw | 更高吞吐 / 96-bit nonce | 自管 nonce，错用即灾难 | ❌ 工作负载 < 1k 行不需要 |
| `nacl.SecretBox` | XSalsa20-Poly1305 / 极简 API | 又引一个加密库依赖 | ❌ 不必要的多样化 |
| SQLCipher | 整库加密 | 替换 sqlite3 driver / 侵入 base.py | ❌ 与 Out-of-Scope 一致 |

依赖新增：`cryptography>=41,<46`

### 3.2 Master key 来源：环境变量 `BIAGENT_MASTER_KEY`

- **格式**：Fernet 标准 32 字节 base64-urlsafe（`Fernet.generate_key()` 输出）
- **来源**：`os.environ["BIAGENT_MASTER_KEY"]` **唯一**（不读 .env auto-load、不读 app_settings 避鸡生蛋）
- **缺失** → 抛 `ConfigMissingError("BIAGENT_MASTER_KEY 未设置")`，**严禁**自动生成
- **格式错误** → 抛 `ConfigMissingError("BIAGENT_MASTER_KEY 格式无效，需 Fernet base64-urlsafe 32 字节")`

### 3.3 加密产物前缀：`enc_v1:`

- 写：`enc_v1:` + `Fernet(key).encrypt(plaintext.encode()).decode()`
- 读：见 `enc_v1:` strip 走 decrypt；无前缀视为未迁移老明文（迁移期透明放行 + INFO log）
- 空字符串 `""` → 短路返 `enc_v1:`（占位区分 NULL）；decrypt `enc_v1:` 反向返 `""`

```python
# adapters/crypto/base.py
from typing import Protocol, runtime_checkable

@runtime_checkable
class CryptoAdapter(Protocol):
    def encrypt(self, plaintext: str) -> str: ...
    def decrypt(self, ciphertext: str) -> str: ...
    def is_encrypted(self, s: str) -> bool: ...

# adapters/crypto/fernet.py
import os
from functools import lru_cache
from cryptography.fernet import Fernet, InvalidToken
from bi_agent.models.errors import ConfigMissingError

ENC_PREFIX = "enc_v1:"
_ENV = "BIAGENT_MASTER_KEY"


class FernetAdapter:
    def __init__(self, master_key: bytes):
        self._fernet = Fernet(master_key)

    def encrypt(self, plaintext: str) -> str:
        if plaintext is None: return ""
        if plaintext == "": return ENC_PREFIX
        return ENC_PREFIX + self._fernet.encrypt(plaintext.encode("utf-8")).decode("ascii")

    def decrypt(self, ciphertext: str) -> str:
        if not ciphertext: return ""
        if not ciphertext.startswith(ENC_PREFIX):
            return ciphertext  # 老明文兼容
        body = ciphertext[len(ENC_PREFIX):]
        if body == "": return ""
        try:
            return self._fernet.decrypt(body.encode("ascii")).decode("utf-8")
        except InvalidToken as e:
            raise ConfigMissingError("解密失败 — master key 可能已变更或密文损坏") from e

    def is_encrypted(self, s: str) -> bool:
        return isinstance(s, str) and s.startswith(ENC_PREFIX)


@lru_cache(maxsize=1)
def get_crypto_adapter() -> CryptoAdapter:
    """R-40：进程内单例；测试 fixture 调 .cache_clear()。"""
    raw = os.environ.get(_ENV)
    if not raw:
        raise ConfigMissingError(f"{_ENV} 未设置 — v0.4.5 必须配置主密钥")
    try:
        return FernetAdapter(raw.encode() if isinstance(raw, str) else raw)
    except (ValueError, TypeError) as e:
        raise ConfigMissingError(f"{_ENV} 格式无效（需 Fernet base64-urlsafe 32 字节）: {e}") from e


def assert_master_key_loaded() -> None:
    """启动期主动校验。"""
    get_crypto_adapter()
```

`adapters/crypto/__init__.py` 导出便捷顶层函数 `encrypt(s)` / `decrypt(s)` / `is_encrypted(s)` 内部走 `get_crypto_adapter()`，方便 repositories 一行 import。

_USER_ENCRYPTED_COLS = ("api_key", "openrouter_api_key", "embedding_api_key", "doris_password")


def _decrypt_user_row(row: dict | None) -> dict | None:
    if row is None: return None

### 4.3 迁移脚本 `bi_agent/scripts/migrate_encrypt_v045.py`（NEW）

```python
"""一次性迁移：明文 6 类敏感列 → enc_v1: 密文。
执行：python3 -m bi_agent.scripts.migrate_encrypt_v045 [--dry-run]
R-36 幂等：见 enc_v1: 跳过；中断恢复无副作用。
"""
import argparse, sys
from bi_agent.repositories.base import get_conn
from bi_agent.adapters.crypto import encrypt, is_encrypted
from bi_agent.adapters.crypto.fernet import assert_master_key_loaded
from bi_agent.core.logging_setup import logger

# (table, id_col, sensitive_cols, where_clause)
TARGETS = [
    ("users", "id", ["api_key", "openrouter_api_key", "embedding_api_key", "doris_password"], None),
    ("data_sources", "id", ["db_password"], None),
    ("app_settings", "key", ["value"], "key IN ('openrouter_api_key', 'embedding_api_key')"),
]


def migrate(dry_run: bool = False) -> dict:
    assert_master_key_loaded()
    conn = get_conn()
    stats = {"scanned": 0, "encrypted": 0, "skipped": 0}
    for table, id_col, cols, where in TARGETS:
        sql = f"SELECT {id_col}, {','.join(cols)} FROM {table}"
        if where: sql += f" WHERE {where}"
        for row in conn.execute(sql).fetchall():
            stats["scanned"] += 1
            updates = {}
            for c in cols:
                v = row[c]
                if v is None or v == "" or is_encrypted(v):
                    continue
                updates[c] = encrypt(v)
            if updates and not dry_run:
                set_clause = ",".join(f"{c}=?" for c in updates)
                conn.execute(f"UPDATE {table} SET {set_clause} WHERE {id_col}=?",
                             (*updates.values(), row[id_col]))
                conn.commit()
                stats["encrypted"] += 1
            elif updates:
                stats["encrypted"] += 1  # dry-run 计数
            else:
                stats["skipped"] += 1
    logger.info(f"migrate_encrypt_v045 完成: {stats}")
    return stats


if __name__ == "__main__":
    ap = argparse.ArgumentParser()
    ap.add_argument("--dry-run", action="store_true")
    args = ap.parse_args()
    sys.exit(0 if migrate(args.dry_run)["scanned"] >= 0 else 1)
```

### 4.5 前端 masked（R-39）

**API 层**：
- GET 路由返回敏感字段时 → `mask_secret(plaintext)` 输出（`••••••••last4`）
- PATCH 路由策略：
  - 字段缺失 → 不更新
  - 字段为 `""` → **保留原值**（用户编辑表单空白即「不改」）
  - 字段为 mask 占位（`••••••••...`）→ **保留原值**（误回传兜底）
  - 字段为新明文 → 更新

```python
def mask_secret(s: str) -> str:
    if not s: return ""
    if len(s) <= 4: return "••••"
    return "•" * 8 + s[-4:]
```

**前端**：input placeholder `••••••••last4`；空白提交 = 不改；点「修改」清空 input 等待新值。
