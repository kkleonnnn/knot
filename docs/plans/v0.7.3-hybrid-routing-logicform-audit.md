## 0. 一句话定位

> v0.7.1/.2 让"已定义指标"走确定性编译（命中→语义层 / 未命中→LLM 兜底），但 **LogicForm 算完即丢、用户/admin 看不见走了哪条路、AI 理解错了无从修正**。
> **v0.7.3 让 LogicForm 浮出水面** —— ① **持久化 + 路由可观测**（哪条路 + 何 LogicForm）② **admin 审计屏**（看 AI 如何理解每个查询）③ **修正**（"AI 理解成 city，其实你要 channel" → 改 LogicForm re-run，比改 SQL 友好 10 倍）。

⚠️ **保守铁律延续**：混合路由的**命中判定 gate 不放松**（v0.7.1/.2 的「parse+compile 成功 → 用，否则回退」binary gate sustained）。v0.7.3 是**可观测 + 可审计 + 可修正**，**非**改路由风险面。
