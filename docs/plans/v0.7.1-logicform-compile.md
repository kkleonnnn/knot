## 7. 验收标准

- **确定性**：同 LogicForm 编译 2 次 → byte-equal SQL（R-SL-17）。
- **单对象**：多对象 LF → 编译 raise → F4 回退 LLM（R-SL-18）。
- **混合回退**：未定义指标 / 解析失败 / flag off → 走现有 `arun_sql_agent`，**零定义查询 byte-equal 现状**（R-SL-14/20）。
- **async**：semantic/ grep 0 sync LLM；解析走 `_allm` R-26 gate（R-SL-15）。
- **脱敏**：非 admin 语义层结果 surface 0 裸表名（R-SL-16）。
- **OOS-1**：编译只取 active catalog metric/tables；multi-catalog 隔离（R-SL-21）。
- **笛卡尔积**：编译产出过 `is_cartesian`/`_is_fan_out`。
- **CI 全绿**：lint-test（pytest + import-linter **8 contracts 不变**）+ frontend-lint + boot smoke 0.7.1 + docker build。
- **flag off 端到端**：== 现有路径（0 行为变更）。
